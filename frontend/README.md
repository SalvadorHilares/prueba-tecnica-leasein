# Frontend LeaseIn - Vue 3

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El frontend estará disponible en `http://localhost:3000`

## Build para producción

```bash
npm run build
```

## Características

- Vue 3 con Composition API
- PrimeVue DataTable con filtros por columna
- Tailwind CSS para estilos
- Filtrado por códigos múltiples (textarea)
- Validación de códigos con API Laravel
- Alertas visuales para códigos no encontrados
- Diseño responsive
